# e-plantShopping

## Paradise Nursery - Plant Shopping Application

**e-plantShopping** is a modern e-commerce web application for purchasing plants online. This repository contains a full-stack shopping cart application built with React and Redux Toolkit.

### Project Overview

Paradise Nursery is a beautiful and user-friendly plant shopping platform that offers a wide variety of plants organized into different categories. The application provides an intuitive shopping experience where users can browse through plant collections, add items to their cart, and manage their purchases seamlessly.

### Features

- **Landing Page**: Welcoming homepage with a "Get Started" button to begin shopping
- **Product Catalog**: Browse plants organized by categories:
  - Air Purifying Plants
  - Aromatic Fragrant Plants
  - Insect Repellent Plants
  - Medicinal Plants
  - Low Maintenance Plants
- **Shopping Cart**: Add, remove, and manage items in your cart
- **Responsive Design**: Modern UI with smooth transitions and animations
- **State Management**: Built with Redux Toolkit for efficient state management

### Technology Stack

- **React** - Frontend framework
- **Redux Toolkit** - State management
- **Vite** - Build tool and development server
- **CSS3** - Styling and animations

### Getting Started

#### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

#### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/e-plantShopping.git
cd e-plantShopping
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
